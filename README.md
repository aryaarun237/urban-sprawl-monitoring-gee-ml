# Urban Sprawl Monitoring and Prediction using GEE and Machine Learning

## Overview
This project monitors and predicts urban sprawl in Chennai, Hyderabad, and Bengaluru using:

- GHSL built-up datasets
- Sentinel-2 imagery
- Google Earth Engine
- Random Forest
- Polynomial Regression

## Features
- Multi-temporal urban growth analysis
- Built-up area extraction
- Spectral index computation
- 2030 urban prediction
- Sprawl visualization maps

## Technologies Used
- Python
- Google Earth Engine
- Rasterio
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Results
- Achieved R² up to 0.834
- Generated 2030 urban expansion predictions
- Compared Polynomial vs Random Forest models
